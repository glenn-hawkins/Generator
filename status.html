Welcome to Generator

Generator is a database driven site meant to dynamically format lists according to user specifications. A record of updates to the site follows.

v0.92b(restored) - 2011/10/10

Site is back online. However, the database must be rebuilt from scratch. Streamlined the updating procedure by resetting focus after an entry is submitted.

v0.92b - 2011/2/17

The practice sheet and flash card templates were updated to properly handle entries that had no 漢字 value. Dynamically generated page titles were added to all of the templates.

v0.91b - 2011/2/2

Generator beta begins today. So far only two chapters have been fully entered into the database, BK Lesson 1 and FSJ Lesson 9. Beta volunteers will be recruited to assist with data entry, with the goal of having every chapter in every Japanese book used at Sophia in the database. The site will go live once an acceptable amount of data has been entered.

Before the site leaves beta, the update page will be protected. Currently it is open to anyone that stumbles across it, leaving the integrity of the database quite vulnerable. Authentication and a method to prevent injection attacks will be implemented. Optimizations will also be made to the database and generated documents, potentially including the page titles that were dropped yesterday and print-media specific styles to improve cross-browser print consistency.

v0.9a - 2011/2/1

Template pages have been built and are functional, although dynamically generated page titles were sacrificed barring a complete database redesign or an understanding of mysqli_multi_query(). Code on all pages has been cleaned up. In this last alpha version, the structure of the site is completed and ready for review.

v0.3a - 2011/1/31

Update and Generator pages are now functional. The code needs to be cleaned up, but first the template pages for Generator to call need to be built. Functionality at a minimum level will be established before returning to cosmetics.

v0.21a - 2011/1/29

Testing has revealed the source of the encoding problem, which has been resolved. Values now display correctly when called from the database.

v0.2a - 2011/1/27

Database connectivity has been established on the update page, however Generator is experiencing difficulties with encoding Japanese characters into the database. At some point during the connection, the characters are switched from UTF8 encoding to latin, creating a complete mess. This is a problem that needs sorting.

v0.1a - 2011/1/26

Generator has been formatted and styled, with the most basic features and the database established. The site uses HTML, CSS, JavaScript, and PHP to interact with a MySQL database. Currently the only database driven feature to be implemented is the select boxes on the "Update" and "Generator" pages.
